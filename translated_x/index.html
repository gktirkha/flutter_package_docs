translated_x

translated_x is a Dart library that provides an extension on the String class for easy translation and customization of text widgets in Flutter applications. It simplifies the creation of Text widgets with translation capabilities and customizable margins.

This Library is designed and tested only on get (getx state management) translation

Features

  • Translation Integration: Easily use translated strings in your Flutter app.
  • Customizable Margins: Add margins around the text widget with top, bottom, left, and right margin options.
  • Text Customization: Customize various text properties including style, alignment, overflow, and more.

Installation

Add translated_x to your pubspec.yaml file:

dependencies:
  translated_x:
    git: https://github.com/gktirkha/translated_x/

If you prefer you can read the dart doc version of the documentation here

Usage

Import the library and use the translatedX extension method on any String:

import 'package:translated_x/translated_x.dart';
import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('translated_x Example')),
        body: Center(
          child: 'hello'.translatedX(
            style: TextStyle(fontSize: 20),
            textAlign: TextAlign.center,
            topMargin: 10.0,
            bottomMargin: 10.0,
          ),
        ),
      ),
    );
  }
}

Parameters

  1. style: The text style to apply to the text.
  2. overflow: How visual overflow should be handled.
  3. maxLines: The maximum number of lines to display.
  4. textAlign: How the text should be aligned horizontally.
  5. locale: The locale to use for translation.
  6. selectionColor: The color of the selection handle.
  7. semanticsLabel: A description of the text used by accessibility tools.
  8. softWrap: Whether the text should break at soft line breaks.
  9. strutStyle: The strut style to apply to the text.
  10. textDirection: The direction in which the text should be laid out.
  11. textHeightBehavior: Defines how text height should be calculated.
  12. textScaleFactor: Deprecated parameter for scaling text size.
  13. textWidthBasis: How the width of the text should be calculated.
  14. textScaler: A custom text scaler to adjust text sizes.
  15. topMargin: The margin to apply at the top of the text widget.
  16. bottomMargin: The margin to apply at the bottom of the text widget.
  17. leftMargin: The margin to apply on the left side of the text widget.
  18. rightMargin: The margin to apply on the right side of the text widget.

Note: The textScaleFactor parameter is deprecated and should be avoided.
