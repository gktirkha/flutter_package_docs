Wraps the widget with a Container widget with specific margin values for each side.

left specifies the margin on the left side. top specifies the margin on the top side. right specifies the margin on the right side. bottom specifies the margin on the bottom side.

Example:

Text('Hello').marginOnly(left: 8.0, top: 16.0);

Implementation

Widget marginOnly({
  double left = 0.0,
  double top = 0.0,
  double right = 0.0,
  double bottom = 0.0,
}) =>
    Container(
        margin: EdgeInsets.only(
            top: top, left: left, right: right, bottom: bottom),
        child: this);
