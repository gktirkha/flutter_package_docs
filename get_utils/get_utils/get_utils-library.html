get_utils library

The get_utils library provides various utilities and extensions extracted from the GetX package. These utilities facilitate common tasks and platform-specific operations, making them easier to use and more accessible.

This library exports the following modules:

  • src/extensions/export.dart: Extensions that add useful methods to existing classes.
  • src/get_utils/get_utils.dart: Core utility functions.
  • src/platform/platform.dart: Platform-specific utilities.
